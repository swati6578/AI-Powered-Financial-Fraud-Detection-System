# AI-Powered Financial Fraud Detection System

An AI-driven web application designed to protect modern digital banking systems from malicious activities through real-time predictive analytics and anomaly detection. Built with Python, Streamlit, and Machine Learning algorithms.

## Objective
The goal of this project is to automate the identification of fraudulent banking transactions, reducing financial risk and providing real-time intelligence to risk management teams.

## Key Features
- **Real-Time Risk Scoring:** Automatically evaluates threat metrics for all incoming transactions.
- **Anomalous Pattern Flagging:** Instantly detects spikes in transaction velocity or unfamiliar geolocations.
- **Interactive Admin Dashboard:** A clean visual panel showing volatile account flags, fraud graphs, and status metrics.

## Technology Stack
- **Language:** Python
- **Libraries:** Scikit-Learn, XGBoost, Pandas, NumPy, Imbalanced-Learn (SMOTE)
- **Frontend Framework:** Streamlit
